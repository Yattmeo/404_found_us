context_mean  # Zero-centered!\n\n# Prediction:\nprediction = merchant_context_mean + learned_residual\n\n# Issue: Residuals are by definition centered around 0\n# Any non-zero prediction → worse than baseline\n# Can't beat: context_mean + 0 = context_mean\n```\n\n**Why This Fails**:\n1.
